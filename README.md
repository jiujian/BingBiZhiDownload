# 必应每日壁纸下载器

一个拥有图形用户界面（GUI）的桌面应用程序，用于下载微软必应的每日高清壁纸。

## 功能特性

- ✅ **图形用户界面**: 直观易用的桌面应用，基于 tkinter 开发
- ✅ **灵活的日期选择**: 支持下载最近N天、指定单日或日期范围的壁纸
- ✅ **多种分辨率**: 支持 1920x1080、4K UHD 等多种分辨率选择
- ✅ **多线程下载**: 可设置并发线程数（1-20），加快下载速度
- ✅ **区域市场支持**: 支持不同地区的必应壁纸（中国、美国、日本等）
- ✅ **智能归档**: 可选择按年份/月份自动创建子目录归档
- ✅ **重复检查**: 自动跳过已存在的文件，避免重复下载
- ✅ **实时日志**: 实时显示下载进度和状态信息

## 系统要求

- Python 3.6 或更高版本
- Windows / macOS / Linux（支持 tkinter 的系统）

## 安装步骤

1. **克隆或下载项目到本地**

2. **安装依赖包**:
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

1. **启动程序**:
   ```bash
   python main.py
   ```

2. **配置下载选项**:
   - 选择日期模式（最近N天/指定单日/日期范围）
   - 选择输出目录
   - 设置分辨率（默认 1920x1080）
   - 选择区域市场（默认 中国）
   - 设置线程数（默认 5）
   - 选择是否按年月归档

3. **开始下载**:
   - 点击"开始下载"按钮
   - 在日志窗口查看下载进度

## 功能说明

### 日期选择模式

- **下载最近**: 下载最近N天的壁纸（例如最近7天）
- **指定单日**: 下载特定某一天的壁纸（格式: YYYY-MM-DD）
- **日期范围**: 下载一个日期范围内的所有壁纸

### 分辨率选项

- 1920x1080 (默认)
- UHD (4K)
- 1366x768
- 1024x768
- 800x600
- 原始尺寸

### 区域市场

支持以下区域市场：
- 中国 (zh-CN)
- 美国 (en-US)
- 日本 (ja-JP)
- 英国 (en-GB)
- 德国 (de-DE)
- 法国 (fr-FR)

### 文件命名规则

下载的文件命名格式: `YYYY-MM-DD_ImageTitle.jpg`

例如: `2024-01-15_Beautiful_Sunset.jpg`

### 归档功能

启用"按年月归档"后，文件将保存在如下目录结构中：
```
下载目录/
  ├── 2024/
  │   ├── 01/
  │   │   ├── 2024-01-15_Image1.jpg
  │   │   └── 2024-01-16_Image2.jpg
  │   └── 02/
  │       └── ...
  └── 2023/
      └── ...
```

## 技术栈

- **Python 3.x**: 编程语言
- **tkinter**: GUI 框架（Python 标准库）
- **requests**: HTTP 请求库
- **concurrent.futures**: 多线程下载
- **threading**: 线程管理

## 注意事项

1. 确保网络连接正常，程序需要访问必应官网获取壁纸信息
2. 线程数设置过高可能导致下载失败，建议使用 5-10 个线程
3. 某些历史日期的壁纸可能无法获取（取决于必应API的限制）
4. 文件名中的特殊字符会被自动清理，确保文件名合法
